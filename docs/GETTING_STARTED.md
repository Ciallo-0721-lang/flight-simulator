# Getting Started with Flight Simulator

## Prerequisites

- Python 3.8 or higher
- pip (Python package manager)
- Git

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/Ciallo-0721-lang/flight-simulator.git
cd flight-simulator
```

### 2. Create Virtual Environment (Optional but Recommended)

```bash
python -m venv venv

# On Windows
venv\Scripts\activate

# On macOS/Linux
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

## Running the Game

```bash
python src/main.py
```

The game window should appear with a basic flight scenario.

## Controls

### Flight Control
- **W / S**: Pitch up / Pitch down
- **A / D**: Roll left / Roll right
- **Q / E**: Yaw left / Yaw right
- **Space**: Increase throttle
- **Ctrl**: Decrease throttle
- **R**: Reset aircraft
- **Esc**: Quit game

### Joystick (Optional)
- Left stick: Pitch and roll
- Right triggers: Throttle
- Buttons: Reserved for future use

## Understanding the Flight Model

### Key Concepts

1. **Lift:** Upward force generated by wings
   - Increases with airspeed and angle of attack
   - Limited by stall angle

2. **Drag:** Resistance to motion
   - Increases with speed (quadratically)
   - Increases with lift (induced drag)

3. **Thrust:** Forward force from engines
   - Controlled by throttle
   - Limited by aircraft capability

4. **Angle of Attack:** Angle between wing and airflow
   - Affects lift coefficient
   - Too high causes stall

### Flight Dynamics

The aircraft motion follows Newton's second law:

```
F = ma (linear motion)
τ = Iα (rotational motion)
```

Forces acting on the aircraft:
1. Lift (L)
2. Drag (D)
3. Thrust (T)
4. Weight (W)

## Common Tasks

### Taking Off

1. Set throttle to maximum (Space key)
2. Maintain level attitude (don't pitch up too early)
3. When airspeed reaches ~50 m/s, gently pull back on stick (S key)
4. Climb to desired altitude

### Landing

1. Reduce throttle gradually (Ctrl key)
2. Maintain descent rate
3. Keep wings level (A/D keys)
4. Flare (pitch up slightly) just before touchdown
5. Come to a stop

### Stable Flight

1. Set throttle to cruise (around 0.6-0.7)
2. Trim aircraft (currently manual - adjust elevator)
3. Monitor altitude and airspeed
4. Make small, smooth control inputs

## Troubleshooting

### Game won't start

- Ensure Python 3.8+ is installed
- Check all dependencies are installed: `pip install -r requirements.txt`
- Check for error messages in `flight_simulator.log`

### Aircraft is unstable

- Make smooth, gradual control inputs
- Don't pull back on stick too hard (can cause stall)
- Check that throttle is at appropriate level for current conditions

### Low frame rate

- Reduce graphics settings in `config/settings.yaml`
- Close other applications
- Check GPU drivers are up to date
